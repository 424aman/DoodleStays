# Beach House Guest Site

A static guest guide website for a six-bedroom beachfront vacation rental.

## Open it in VS Code

1. Extract the ZIP file.
2. Open the `airbnb-guest-site` folder in VS Code.
3. Install the **Live Server** extension.
4. Right-click `index.html`.
5. Select **Open with Live Server**.

## Add your own photos

Put your images inside the `images` folder using these filenames:

- `hero.jpg`
- `ocean-view.jpg`
- `activities.jpg`
- `kitchen.jpg`
- `faq.jpg`
- `contact.jpg`
- `fridge.jpg`
- `oven.jpg`
- `coffee.jpg`
- `dishwasher.jpg`
- `microwave.jpg`

You may also change the filenames in `style.css`.

## Customize the information

Search `index.html` for these placeholders:

- `Your Host Name`
- `(555) 555-5555`
- `host@example.com`
- Sample check-in and check-out times
- Sample restaurant names
- Appliance instructions
- Parking and beach-access instructions

## Deploy to Vercel

Use these settings:

- Framework Preset: `Other`
- Root Directory: `./`
- Build Command: leave blank
- Output Directory: leave blank
- Install Command: leave blank

Vercel will serve `index.html` directly.
